# 🏋️ Powerlifting Training App

A web application for powerlifting athletes that provides:

* 📋 Structured training programs (bench, squat, deadlift)
* 📊 Athlete level calculation based on standards
* 📈 Progress tracking and training sessions

---

## 🚀 Tech Stack

* Next.js
* React
* TypeScript
* SCSS Modules
* Zustand (state management)

---

## 📦 Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

---

## 🧠 Features

* Training program system (weeks → days → exercises)
* Dynamic workout rendering
* Athlete standards calculation
* Responsive UI (desktop + mobile)
* Sticky and floating UI interactions
